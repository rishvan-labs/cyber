# Methodology Explanation

## Monitor Mode
Monitor mode allows the wireless adapter to capture all packets in the air.

## Network Scanning
Used to identify available Wi-Fi networks, channels, and encryption types.

## Handshake Capture
Capturing handshake packets is essential for WPA/WPA2 analysis.

## Password Testing
Wordlists are used to test password strength.

## Security Insight
Even strong encryption fails if passwords are weak.
